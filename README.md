# Productivity Tracker (Python)

## Overview
This is a command-line based productivity tracker built using Python.
It allows users to log study sessions and view productivity analytics.

## Features
- Add study sessions (subject + duration)
- Persistent data storage using JSON
- Total study summary
- View sessions by specific date
- Weekly study summary (last 7 days)
- Input validation and error handling

## Project Structure
productivity_tracker/
├── main.py
├── tracker.py
├── storage.py
├── utils.py
└── data.json

## How to Run
1. Make sure Python is installed
2. Open terminal in project folder
3. Run:
   python main.py
